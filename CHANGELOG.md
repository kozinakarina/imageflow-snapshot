# Changelog

## [Snapshot] - 2025-10-31

### Added
- Градиент поднят выше (transition_start=360, transition_end=960)
- Увеличен blur_size до 300 для более плавного перехода
- Текст игры поднят на 34px от провайдера
- Автоматическое разделение названий игр пробелами (split_game_title)
- Подтверждено извлечение доминантных цветов строго из фона

### Features
- Seedream API интеграция для очистки изображений
- RMBG удаление фона с моделью u2net
- Инпейнтинг с masked blur только для фона
- Извлечение цветов через KMeans с маской фона
- Градиентная композиция с плавной маской
- Текстовые оверлеи с автоматическим форматированием
- FastAPI сервис с поддержкой Nginx

### Technical Details
- Python 3.x
- FastAPI + Uvicorn
- PIL/Pillow для обработки изображений
- OpenCV для инпейнтинга и размытия
- scikit-learn для KMeans
- rembg для удаления фона
